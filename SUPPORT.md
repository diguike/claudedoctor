# Support

- Use a bug report for reproducible Claude Doctor failures or incorrect verdicts.
- Use a feature request for new detectors, platforms, or output formats.
- Include `claudedoctor env --json` only after reviewing the sanitized output. Remove file paths, public IPs, organization names, and any other data you do not want to publish.
- Do not post credentials or `~/.claude/.credentials.json` contents.

Claude Doctor is an independent open-source project and is not affiliated with or endorsed by Anthropic. It cannot inspect Anthropic's private enforcement systems, recover an account, or explain a specific suspension. Account and billing issues belong with official Anthropic support.
